Announcement April 14, 2026

Season 12 Has Begun

Pinnacle SMP Season 12 is officially live! A brand-new world has been generated, giving everyone a fresh start to explore, build, and grow together.

To keep progression balanced and the community close-knit early on, a world border has been set at 3,000 blocks from spawn. This ensures:

  • A more active and connected player base.
  • Fair access to resources for everyone.
  • Stronger early-season collaboration.

The border may expand later in the season, but for now, it’s all about making the most of what’s inside. Jump in, get established, and be part of the beginning of something great. Welcome to Season 12.

Announcement April 6, 2026

Update Still In Progress

"Alpha builds for Paper 26.1.1 are now available. There is still no ETA for a stable build." - Paper Developers. This means they are now going into testing for the update, but we still do not know when we will be able to update our server to the newest version of Minecraft.

Remember this update is a really big one in terms of how servers are run for two main reasons:

  • The source code for Minecraft was unobfuscated, meaning old ways of working around obfuscation are now creating problems. This affects all mods and server clients and will take time to work around, but in the long term it should make things easier and more stable because developers can code against what is actually there instead of having to guess.
  • Mojang also made changes to how world save data is stored at the same time as making the source code unobfuscated. This means update teams have to be extra careful so worlds do not corrupt when moving between versions.

When there are any more updates, I will let everyone know.

Announcement March 24, 2026

Ranks System Update

We've updated how ranks work to better reflect activity, contribution, and community involvement rather than only time played. Promotions are no longer automatic or purely time-based.

What changed

  • More weight on positive community impact and collaboration.
  • Staff review is now part of rank advancement.
